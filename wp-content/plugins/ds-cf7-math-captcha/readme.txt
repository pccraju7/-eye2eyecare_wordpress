"DS CF7 Math Captcha" is a math captcha with refresh captcha functionality to prevent unwanted spam for your contact form 7 plugin.

== Description ==

### Form Stop Spam Emails - A permanent solution to stop spam emails

"DS CF7 Math Captcha" plugin gives functionality of prevent unwanted spam to your contact form 7. Contact Form 7 plugin is most widely used by many users so this plugin helps to prevent unwanted spam from website that has been created by Contact Form 7 plugin. This plugin is very much effective to stop form spam for Contact Form 7 plugin.

= The plugin has been tested with the latest version (6.0) of Contact Form 7 =

= Features =
 * Math Captcha
 * Refresh Math Captcha
 
== Installation ==

In most cases you can install plugin by single click from WordPress admin dashboard.

However, if you install this manually, follow these steps:

1. Upload the entire `ds-cf7-math-captcha` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

#How I can add captcha in contact form 7?
To add the mathcaptcha you can find the tag 'Mathcaptcha' when you will create new contact form. When you will click on 'Mathcaptcha' tag then that tag shortcode will display in the form editor. That's all!!
